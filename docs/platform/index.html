Extends

プラットホームの機能を拡張するアドオン群、アドオン名や簡単な説明がついています。

YIC-Mk_EX-platform-1f_1 / 同-3f_2

このアドオン開発はあるユーザーからの「拡幅ホーム」というキーワードから始まりました。
最初に実装したのは拡幅対応の為に用意されたこれらのアドオンです。

YIC-Mk_EX-platform-1_Closed

JR京都・神戸線の車窓動画を見ていて思いついたので作成。
特急や優等街道な路線に投入したり、8両編成の運行を予定していたが4両編成しか走らない地下鉄線のホームなどに使えそうです。

YIC-Mk_EX-platform-1u_LR

構内踏切のアドオンはデフォルトで実装されていますが、両面ホーム対応というニッチ極まりない需要が湧いたので作成。

YIC-Mk_EX-platform-1u_Ef

構内踏切アドオン第2弾、フェンスが付いたので駅舎と隣接しないホームや長さが違うホームもサポート

YIC-Mk_EX-platform-1u-ExF

構内踏切アドオン第3弾、中線を渡っていってね!

Wides

プラットホームを拡幅するためのアドオン群

YIC-Mk_EX-platform-1f_T1-1

拡幅対応版ホーム、既存のホームを埋めて拡幅しましたと言えるように。
こっちは常設ですが西武新宿線の沼袋駅がモチーフだったり。

YIC-Mk_EX-platform-0_Wide

田舎の駅だけど昔は大ターミナルだったサムシングな感覚で作成

YIC-Mk_EX-platform-2f_Tw / YIC-Mk_EX-platform-3f_Tw

ホームの屋根を広げたかったので作成、まだ未完だったことが判明した。
